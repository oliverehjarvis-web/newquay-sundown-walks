# Newquay Sundown Walks - Website

This is the official website for Newquay Sundown Walks, a specialist evening and late-night dog walking service based in Newquay, Cornwall.

## Tech Stack
-   HTML5
-   CSS3
-   Hosted on GitHub Pages
-   DNS by Cloudflare

## Purpose
This is a simple, single-page static site designed to:
-   Build trust with potential clients.
-   Rank for local SEO terms like "evening dog walker newquay".
-   Provide a clear point of contact.
